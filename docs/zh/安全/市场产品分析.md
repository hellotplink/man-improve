## 产品风险等级

### 风险分级说明

| 级别 | 说明 |
|------|------|
| 🟢 低风险 | 正规处方药，医生指导下使用 |
| 🟡 中等风险 | 正规渠道的经验证仿制药 |
| 🔴 高风险 | 未经验证的单一成分产品 |
| ⚫ 极高风险 | 复方产品、来源不明产品 |

### 按产品类型分类

#### 口服果冻/液体剂型
- **风险等级**：⚫ 极高
- 无法验证剂量
- 容易受污染
- 成分可能与标注不符

#### 泡腾片/溶解片
- **风险等级**：🔴 高
- 剂型不稳定
- 剂量不可靠
- 可能含有害添加剂

#### 复方产品
- **风险等级**：⚫ 极高
- 固定剂量不适合大多数人
- 副作用可能叠加
- 药物相互作用风险

## 安全替代方案

### 合法获取途径

1. **正规医院**：获得处方后在医院药房购买
2. **持证药店**：凭处方在正规连锁药店购买
3. **互联网医院**：正规平台在线问诊开方
4. **患者援助计划**：部分厂商提供优惠政策

### 经济考虑

- 仿制药已经合法上市，价格合理
- 医保可能覆盖部分费用
- 长期来看，安全用药更经济

## 产品识别要点

### 可能的问题产品特征

#### 外观检查
- 颜色或形状异常
- 印刷质量差
- 无正规包装
- 药片外观不一致

#### 包装检查
- 无批号
- 无有效期
- 拼写错误
- 无厂商地址
- 宣称夸大

#### 来源检查
- 无需处方即可购买
- 通过社交媒体销售
- 价格异常低廉
- 从意外国家发货
- 无正规药房参与

## 紧急情况处理

### 如果使用了来源不明的产品

#### 立即采取的措施
1. **记录时间**和服用量
2. **保留包装**和剩余产品
3. **不要自行催吐**除非医生指导
4. **联系急救**或中毒咨询
5. **如实告知**医护人员

#### 紧急联系方式
- **中国大陆**：120
- **中毒咨询**：当地疾控中心
