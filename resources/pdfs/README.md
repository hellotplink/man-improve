## 阅读建议

### 入门路线

```
第一阶段：了解基础
├── 《情商》- 了解情绪管理
├── 《非暴力沟通》- 学习沟通方法
└── 《爱的五种语言》- 理解爱的表达

第二阶段：深入学习
├── 《亲密关系》- 理解关系本质
├── 《幸福的婚姻》- 经营长期关系
└── 《当下的力量》- 心灵成长

第三阶段：专业知识
├── 《金赛性学报告》- 性学基础
├── 《运动改造大脑》- 运动科学
└── 中医养生相关书籍
```

### 阅读方法

1. **不求快**：慢慢读，做笔记
2. **实践为王**：读一点，做一点
3. **反复阅读**：经典值得多读几遍
4. **交流讨论**：和伴侣一起读
