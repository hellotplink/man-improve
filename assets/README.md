# 图片资源

> 项目中使用的图片资源

## 目录结构

```
assets/
├── diagrams/          # 示意图、流程图
├── infographics/      # 信息图表
├── icons/             # 图标
└── misc/              # 其他图片
```

## 使用方法

在 Markdown 文件中引用图片：

```markdown
![图片描述](./assets/diagrams/example.png)
```

从子目录引用：

```markdown
![图片描述](../assets/diagrams/example.png)
```

## 图片规范

- 格式：优先使用 PNG 或 SVG
- 命名：使用小写英文，单词间用连字符（如 `blood-flow-diagram.png`）
- 大小：建议压缩后不超过 500KB
