## 重要提醒

### ⚠️ 安全警示

1. **仅供学习参考** - 所有内容仅用于教育目的
2. **咨询专业人员** - 使用任何方剂前必须咨询中医师
3. **个体差异** - 每个人体质不同，切勿盲目照搬
4. **药物禁忌** - 注意配伍禁忌和使用禁忌
5. **正规渠道** - 中药材应从正规渠道购买

1. **系统学习** - 不要只看单个方子
2. **理论结合实践** - 先懂理论再应用
3. **记录笔记** - 整理关键知识点
4. **交流讨论** - 与专业人士交流
