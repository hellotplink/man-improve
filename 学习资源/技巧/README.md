## 核心技能概览

### 凯格尔运动

**为什么重要？**
- 增强盆底肌肉力量
- 改善射精控制能力
- 提升勃起质量
- 预防尿失禁

**快速入门：**
1. 找到正确肌肉（排尿时尝试中断）
2. 收缩3秒 → 放松3秒
3. 每天3组，每组10-15次
4. 持续8-12周见效

[👉 查看完整指南](./kegel-exercises.md)

---

### 延时训练

**核心方法：**
- **动-停法**：接近临界点时停止，重复练习
- **挤压法**：在特定位置施加压力
- **呼吸控制**：深呼吸延缓兴奋

**关键原则：**
- 循序渐进，不要急于求成
- 与伴侣沟通配合
- 持续练习4-8周

[👉 查看完整指南](./delay-techniques.md)

### 运动健身

**推荐运动：**

| 运动 | 频率 | 主要益处 |
|------|------|----------|
| 深蹲 | 3次/周 | 刺激睾酮，锻炼盆底 |
| 跑步 | 3-4次/周 | 改善心血管，促进循环 |
| 臀桥 | 3次/周 | 强化臀肌和盆底 |
| 平板支撑 | 每日 | 核心稳定性 |

## 训练建议

### 综合训练计划（8周）

| 周次 | 凯格尔 | 延时练习 | 运动 |
|------|--------|----------|------|
| 1-2 | 每日基础练习 | 动-停法入门 | 徒手深蹲+快走 |
| 3-4 | 增加时长 | 加入挤压法 | 增加强度 |
| 5-6 | 进阶练习 | 伴侣配合 | 增加有氧 |
| 7-8 | 维持训练 | 实践应用 | 综合训练 |

### 成功要素

1. **持之以恒** - 至少坚持8周
2. **循序渐进** - 不要急于求成
3. **记录进步** - 追踪改善情况
4. **综合调理** - 运动+饮食+心理
